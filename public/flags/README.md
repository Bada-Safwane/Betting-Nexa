# Flag Images

Place the country flag PNG images in this directory with the following names:

- `eg.png` - Egypt flag
- `sn.png` - Senegal flag
- `ng.png` - Nigeria flag
- `ma.png` - Morocco flag

## Recommended Sources

You can download flag images from:
- https://flagpedia.net/download
- https://www.countryflags.com/
- https://flagcdn.com/

## Image Specifications

- Format: PNG with transparency
- Recommended size: 256x256px or larger
- Aspect ratio: Standard flag proportions (typically 3:2)
